# VueProject
Online e-store for books developed using vue and bulma

### Used Libraries
- Bulma
- Axios
- Buefy
- Json Server
- Lodash
- NProgress
- Vue Multiselect
- Vue Router
- Vuetable-2
- Vuex
- Vuex Persist
